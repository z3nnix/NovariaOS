# Shell and Filesystem

## Interactive Shell

NovariaOS now includes an interactive shell with keyboard support that allows you to interact with the system in real-time.

### Features
- **Keyboard input**: Full keyboard support with shift and caps lock
- **Command history**: Backspace support for editing
- **Command-line interface**: Traditional shell experience

### Available Commands

#### System Commands
- `help` - Show all available commands
- `clear` - Clear the screen
- `echo <text>` - Echo text back to the console
- `info` - Display system information
- `memtest` - Test memory allocation and deallocation
- `list` - List all programs loaded in initramfs
- `run <index>` - Run a program from initramfs by its index

#### Filesystem Commands
- `ls` - List all files in the virtual filesystem
- `cat <filename>` - Display the contents of a file
- `write <filename> <content>` - Create or overwrite a file with content
- `rm <filename>` - Delete a file from the filesystem

## Virtual Filesystem (VFS)

The VFS is a simple in-memory filesystem that allows you to create, read, and delete files during runtime.

### Specifications
- **Maximum files**: 64 files
- **Maximum filename length**: 32 characters
- **Maximum file size**: 4096 bytes per file
- **Storage**: All data is stored in RAM

### Usage Examples

```
novaria> write hello.txt Hello, NovariaOS!
File 'hello.txt' written successfully

novaria> ls
Files: 1
  hello.txt (17 bytes)

novaria> cat hello.txt
Hello, NovariaOS!

novaria> rm hello.txt
File 'hello.txt' deleted successfully
```

## Keyboard Driver

The keyboard driver supports:
- Standard US keyboard layout
- Shift key for capitalization and special characters
- Caps Lock toggle
- Backspace for editing input
- All printable ASCII characters (32-126)

### Technical Details
- Polling-based input (no interrupts required)
- Circular buffer for keyboard events (256 characters)
- Scancode to ASCII translation tables
