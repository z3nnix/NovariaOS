# Syscalls

| Name          | Number |  Description                              | requires       |
|---------------|--------|-------------------------------------------|----------------|
| EXIT          | 0x00   | process exit                              | -              |
| SPAWN          | 0x01   | spawn program with inheritance caps     | CAP_FS_READ    |
| OPEN          | 0x02   | give file descriptor from file name       | CAP_FS_READ    |
| READ          | 0x03   | file read                                 | CAP_FS_READ    |
| WRITE         | 0x04   | file write                                | CAP_FS_WRITE   |
| CREATE        | 0x05   | file create                               | CAP_FS_CREATE  |
| DELETE        | 0x06   | file delete                               | CAP_FS_DELETE  |
| CAP_CHECK     | 0x07   | caps check via PID                        | -              |
| CAP_SPAWN     | 0x08   | give the caps                             | CAPS_CAPS_MGMT |
| MSG_SEND      | 0x0A   | send message                              | -              |
| MSG_RECV      | 0x0B   | receive message                           | -              |
| PORT_IN_BYTE  | 0x0C   | read byte from I/O port                   | CAP_DRV_ACCESS |
| PORT_OUT_BYTE | 0x0D   | write byte to I/O port                    | CAP_DRV_ACCESS |
