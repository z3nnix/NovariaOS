# Syscalls

| Name          | Number |  Description                              |
|---------------|--------|-------------------------------------------|
| EXIT          | 0x00   | process exit                              |
| EXEC          | 0x01   | execute program with inheritance caps     |
| READ          | 0x02   | file read (require CAP_FS_READ)           |
| WRITE         | 0x03   | file write (require CAP_FS_WRITE)         |
| CREATE        | 0x04   | file create (require CAP_FS_CREATE)       |
| DELETE        | 0x05   | file delete (require CAP_FS_DELETE)       |
| CAP_REQUEST   | 0x06   | caps request (require CAP_CAPS_MGMT)      |
| CAP_SPAWN     | 0x07   | run with caps (require CAPS_CAPS_MGMT)    |
| DRV_CALL      | 0x08   | call some driver (require CAP_DRV_GROUP_*)|
| MSG_SEND      | 0x09   | send message                              |
| MSG_RECIEVE   | 0x0A   | recieve message                           |
